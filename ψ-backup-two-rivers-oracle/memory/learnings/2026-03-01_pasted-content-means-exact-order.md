# Pasted Content Means Exact Order

**Date**: 2026-03-01
**Source**: rrr session — landing page section ordering

## Lesson

When a user pastes the exact visual output from their browser showing the order they want sections in, implement that exact order literally. Don't reinterpret through architectural concepts like "sections" and "components."

## Example

Nat pasted:
```
🌱 ORIGIN STORY
กำเนิดสองแคว...
"Water finds the way..."
แม่น้ำน่าน...
แม่น้ำแควน้อย...
🗺️ เปิดแผนที่เต็มจอ 🚂 ดู 3D Train
```

This meant: origin text → river cards → buttons → then map below. Not "Origin section then Journey section" as separate blocks.

## Principle

When someone is in **visual-arrangement mode**, stay in that mode. Don't propose architecture changes (components, splits) — just move the HTML to match what they see. Two conceptual sections can be one flowing HTML section when they tell a continuous story.
